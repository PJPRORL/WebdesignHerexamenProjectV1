RGB and RGBA Colors

Een RGB-kleurwaarde vertegenwoordigt RODE, GROENE en BLAUWE lichtbronnen. Een RGBA-kleurwaarde is een uitbreiding van RGB met een Alpha-kanaal (dekking).

RGB Color Values

In HTML kan een kleur worden gespecificeerd als een RGB-waarde, met de formule: rgb(rood, groen, blauw). Elke parameter definieert de intensiteit van de kleur met een waarde tussen 0 en 255.

rgb(255, 0, 0)
rgb(0, 0, 255)
rgb(60, 179, 113)
rgb(238, 130, 238)
rgb(255, 165, 0)
rgb(106, 90, 205)
Shades of Gray

Grijstinten worden vaak gedefinieerd met gelijke waarden voor alle drie de parameters:

rgb(60, 60, 60)
rgb(100, 100, 100)
rgb(140, 140, 140)
rgb(180, 180, 180)
rgb(200, 200, 200)
rgb(240, 240, 240)
RGBA Color Values

RGBA-kleurwaarden zijn een uitbreiding van RGB met een Alpha-kanaal dat de dekking specificeert. Een RGBA-kleurwaarde wordt gespecificeerd met: rgba(rood, groen, blauw, alpha). De alphaparameter is een getal tussen 0.0 (volledig transparant) en 1.0 (volledig dekkend).

rgba(..., 0)
rgba(..., 0.2)
rgba(..., 0.4)
rgba(..., 0.6)
