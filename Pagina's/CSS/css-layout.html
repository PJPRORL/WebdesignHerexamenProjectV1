Hoofdstuk: The CSS Box Model

85-88. Box Model Properties

Elk HTML-element kan worden gezien als een doos. Het box model beschrijft de opbouw van deze doos: de content zelf, padding (de ruimte binnen de rand), border (de rand zelf), en margin (de ruimte buiten de rand). Standaard beïnvloeden padding en border de totale grootte van de doos. Het is een best practice om box-sizing: border-box; te gebruiken, zodat de opgegeven width en height inclusief padding en border zijn, wat de layout veel intuïtiever maakt.

.box {
  box-sizing: border-box; /* Zeer aanbevolen! */
  width: 300px; /* Totale breedte is nu 300px */
  height: 150px;
  
  padding: 20px; /* Ruimte binnen de border */
  
  border: 5px solid navy;
  border-radius: 10px; /* Afgeronde hoeken */
  
  margin: 25px; /* Ruimte buiten de border */
}

89. The Display Property

De display eigenschap bepaalt het buitenste en binnenste display-type van een element. Het beïnvloedt de layout fundamenteel.

  • block: Begint op een nieuwe regel, neemt de volledige breedte in.
  • inline: Begint niet op een nieuwe regel, neemt alleen de benodigde breedte in. width en height werken niet.
  • inline-block: Zoals inline, maar width, height en verticale margins/paddings werken wel.
  • none: Het element wordt helemaal niet weergegeven en neemt geen ruimte in.
  • flex en grid: Maken van een element een container voor respectievelijk een flexbox- of grid-layout, wat krachtige manieren zijn om complexe layouts te bouwen.

91-92. CSS Units: em & rem

px (pixels) zijn absolute eenheden. em en rem zijn relatieve eenheden, wat beter is voor schaalbaarheid en toegankelijkheid.

  • em: Relatief aan de font-size van het directe parent element. Dit kan leiden tot onverwachte resultaten bij diep geneste elementen.
  • rem (root em): Relatief aan de font-size van het root () element. Dit is veel voorspelbaarder en wordt sterk aanbevolen voor het instellen van font-sizes, margins en paddings. Als een gebruiker zijn standaard lettergrootte in de browser aanpast, schaalt de hele layout mee.
html {
  font-size: 16px;
}

.parent {
  font-size: 20px;
}

.child-em {
  font-size: 1.5em; /* 1.5 * 20px = 30px */
}

.child-rem {
  font-size: 1.5rem; /* 1.5 * 16px = 24px */
}

Hoofdstuk: Other assorted useful CSS properties

94. Opacity & The Alpha Channel

opacity maakt een heel element doorzichtig, inclusief alle kind-elementen zoals tekst. De waarde gaat van 0 (volledig onzichtbaar) tot 1 (volledig zichtbaar). Het alpha-kanaal in rgba() of hsla() maakt alleen de kleur zelf doorzichtig, terwijl de content van het element (zoals tekst) volledig dekkend blijft. Dit is ideaal voor het creëren van overlays of doorzichtige achtergronden.

.transparent-box {
  opacity: 0.5; /* Element en tekst zijn 50% doorzichtig */
}

.semi-transparent-background {
  /* Alleen de achtergrond is 50% doorzichtig, tekst niet */
  background-color: rgba(0, 0, 255, 0.5); /* Blauw met 50% alpha */
}

95. The Position Property

Hiermee kun je de standaard positionering van een element doorbreken. De coördinaten (top, right, bottom, left) werken alleen op gepositioneerde elementen.

  • relative: Het element is positioneerbaar t.o.v. zijn normale positie, maar blijft in de documentstroom.
  • absolute: Het element wordt uit de normale flow gehaald en gepositioneerd t.o.v. zijn dichtstbijzijnde gepositioneerde voorouder (een voorouder met een andere positie dan static).
  • fixed: Gepositioneerd t.o.v. de viewport (het browservenster). Scrolt niet mee met de pagina. Ideaal voor vaste headers of "terug naar boven"-knoppen.
  • sticky: Een mix van relative en fixed. Gedraagt zich als relative totdat het een bepaalde scrollpositie bereikt, dan wordt het fixed. Perfect voor navigatiebalken.

96-98. Transitions & Transforms

transition zorgt voor een geleidelijke animatie van een eigenschap wanneer deze verandert (bv. bij :hover). Je specificeert welke eigenschap, de duur, en de timing-functie (bv. ease-in-out). transform verandert de vorm, grootte of positie van een element zonder de omliggende elementen te beïnvloeden. Veelvoorkomende transformaties zijn scale(), rotate(), translate() (verplaatsen), en skew() (vervormen).

.fancy-button {
  background-color: blue;
  color: white;
  padding: 1rem 2rem;
  border: none;
  transition: transform 0.3s ease, background-color 0.3s ease;
}

.fancy-button:hover {
  background-color: darkblue;
  transform: translateY(-5px) scale(1.05); /* Beweeg omhoog en maak groter */
}

100. Google Fonts

Voeg eenvoudig honderden gratis, open-source lettertypen toe aan je site. Je kunt ze direct linken via de Google Fonts CDN (Content Delivery Network) of de lettertypebestanden downloaden en zelf hosten. Het linken via CDN is eenvoudiger en vaak sneller, omdat de kans bestaat dat een bezoeker het lettertype al in de cache van zijn browser heeft staan van een andere website.

/* Stap 1: Importeer het lettertype in je CSS (of link in HTML) */
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

/* Stap 2: Gebruik het in je CSS */
body {
  font-family: 'Roboto', sans-serif;
}

Hoofdstuk: Responsive CSS & Flexbox

105. What on Earth Is Flexbox?

Flexbox is een eendimensionaal layout-model dat een efficiënte manier biedt om items in een container uit te lijnen en te verdelen. Het is ontworpen voor het verdelen van ruimte binnen een container en voor het uitlijnen van items, en het lost veel problemen op waar oudere layout-methoden (zoals floats) mee kampten. Je activeert het met display: flex; op de container. Het concept draait om een flex-container en flex-items, met een hoofd-as en een dwars-as.

106-112. Core Flexbox Properties

De meeste eigenschappen worden op de flex container gezet:

  • flex-direction: Bepaalt de hoofd-as (row is horizontaal, column is verticaal).
  • justify-content: Uitlijning op de hoofd-as (bv. flex-start, center, space-between).
  • align-items: Uitlijning op de dwars-as (bv. flex-start, center, stretch).
  • flex-wrap: Bepaalt of items naar een volgende regel mogen springen.

Op de flex items zelf kun je eigenschappen als flex-grow (hoeveel een item mag groeien), flex-shrink (hoeveel het mag krimpen) en align-self (individuele uitlijning op de dwars-as) toepassen.

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center; /* Items horizontaal centreren */
  align-items: center; /* Items verticaal centreren */
  height: 200px;
  background-color: #ddd;
}

113-114. Responsive Design & Media Queries

Responsive design zorgt ervoor dat je layout er goed uitziet op alle apparaten. Media Queries zijn hier de sleutel toe. Ze passen CSS-regels toe op basis van bepaalde voorwaarden van het apparaat, meestal de breedte van de viewport. Een "mobile-first"-aanpak wordt aanbevolen: ontwerp je basisstijlen voor kleine schermen en gebruik dan media queries met min-width om stijlen toe te voegen of aan te passen voor grotere schermen.

.container {
  width: 90%;
  margin: 0 auto;
}

/* Voor schermen die 768px of breder zijn */
@media (min-width: 768px) {
  .container {
    width: 80%;
  }

  body {
    font-size: 18px;
  }
}
