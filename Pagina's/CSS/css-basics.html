Hoofdstuk: Begin van CSS

60. What is CSS?

CSS staat voor Cascading Style Sheets. Het is de taal die we gebruiken om de presentatie en layout van een webpagina te stijlen. Met CSS bepaal je kleuren, lettertypen, afstanden, afmetingen en nog veel meer. Het "Cascading"-gedeelte is belangrijk: het verwijst naar de manier waarop stijlen worden toegepast en conflicten worden opgelost wanneer meerdere regels hetzelfde element targeten.

63-65. Color Systems: Named, RGB, & Hexadecimal

Kleuren in CSS kunnen op verschillende manieren worden gedefinieerd: via benoemde kleuren, RGB(A) en Hexadecimale codes. Naast deze systemen bestaat ook HSL (Hue, Saturation, Lightness), wat vaak intuïtiever is om mee te werken. De 'A' in RGBA en HSLA staat voor Alpha en regelt de transparantie (van 0.0 voor volledig transparant tot 1.0 voor volledig dekkend).

/* Named Color (makkelijk leesbaar, beperkte keuze) */
h1 {
  color: dodgerblue;
}

/* RGB (Red, Green, Blue: waarden van 0-255) */
p {
  color: rgb(100, 150, 200);
}

/* RGBA (met transparantie) */
.overlay {
  background-color: rgba(0, 0, 0, 0.5); /* 50% doorzichtig zwart */
}

/* Hexadecimal (meest gebruikt) */
body {
  background-color: #333333; /* #333 is een shorthand */
}

67-69. Common Text & Font Properties

Met CSS heb je volledige controle over de typografie van je pagina. Naast font-family en font-size, zijn font-weight (dikte), text-align (uitlijning), en line-height (regelafstand) essentieel voor leesbaarheid. Andere nuttige eigenschappen zijn text-transform om tekst in hoofdletters of kleine letters te zetten, en letter-spacing of word-spacing om de ruimte tussen letters of woorden aan te passen.

.my-text {
  font-family: "Arial", sans-serif; /* Fallback lettertype */
  font-size: 16px; /* Grootte in pixels */
  font-weight: bold; /* bold, normal, 700, etc. */
  text-align: center; /* left, right, justify */
  text-decoration: underline; /* none, overline, line-through */
  text-transform: uppercase; /* Maakt tekst HOOFDLETTERS */
  line-height: 1.6; /* Regelafstand */
}

Hoofdstuk: The world of CSS Selectors

71-73. Basic Selectors (Universal, Element, ID, Class)

Selectors worden gebruikt om de HTML-elementen te 'selecteren' die je wilt stijlen. De universele selector (*) is handig voor het toepassen van basis-resets. De element-selector (bv. p) is goed voor algemene stijlen. De ID-selector (#id-naam) is bedoeld voor één uniek element per pagina, terwijl de class-selector (.class-naam) is ontworpen om hergebruikt te worden op meerdere elementen die dezelfde stijl moeten delen.

/* Universal Selector (selecteert alles) */
* {
  box-sizing: border-box;
}

/* Element Selector (selecteert alle p tags) */
p {
  color: #555;
}

/* ID Selector (selecteert het element met id="logo") */
#logo {
  border: 1px solid black;
}

/* Class Selector (selecteert alle elementen met class="highlight") */
.highlight {
  background-color: yellow;
}

74-76. Combinator & Attribute Selectors

Combineer selectors om specifieker te zijn, of selecteer op basis van attributen. Combinators bepalen de relatie tussen selectors. De descendant selector (spatie) is de meest voorkomende. De child selector (>) is specifieker en selecteert alleen directe kinderen. Attribuutselectors zijn zeer krachtig en kunnen elementen selecteren op basis van de aanwezigheid, exacte waarde of zelfs een deel van de waarde van een attribuut.

/* Descendant Selector (selecteert alle p binnen een article) */
article p {
  font-style: italic;
}

/* Direct Child Selector (selecteert li die een direct kind is van ul) */
ul > li {
  list-style-type: square;
}

/* Adjacent Sibling Selector (selecteert de p direct na een h1) */
h1 + p {
  margin-top: 0;
}

/* Attribute Selector (selecteert alle input met type="text") */
input[type="text"] {
  border: 1px solid gray;
}

77-78. Pseudo-Classes & Pseudo-Elements

Pseudo-classes selecteren elementen in een speciale staat (zoals :hover, :focus voor formuliervelden, of :first-child). Pseudo-elementen stijlen een specifiek deel van een element (zoals ::first-letter) of voegen content toe die niet in de HTML staat (zoals ::before en ::after). Deze zijn uiterst nuttig voor UI-feedback en decoratieve effecten zonder extra HTML te hoeven toevoegen.

/* Pseudo-class: stijl een link als de muis eroverheen is */
a:hover {
  color: red;
}

/* Pseudo-class: stijl elk tweede kind-element in een lijst */
li:nth-child(even) {
  background-color: #eee;
}

/* Pseudo-element: stijl de eerste letter van elke paragraaf */
p::first-letter {
  font-size: 2em;
  font-weight: bold;
}

/* Pseudo-element: voeg content toe vóór een element */
.important::before {
  content: "⚠️ ";
}

79-83. Cascade, Specificity & Inheritance

De Cascade bepaalt welke CSS-regel wordt toegepast als er conflicten zijn. Specificity is een weging die de browser toekent aan een CSS-regel om te beslissen welke regel voorrang krijgt (een ID is specifieker dan een klasse, die weer specifieker is dan een element). Inheritance betekent dat sommige CSS-eigenschappen (typografische zoals color, font-family) van een parent-element worden doorgegeven aan de child-elementen. Het !important-statement doorbreekt de cascade en moet alleen als laatste redmiddel gebruikt worden, omdat het de voorspelbaarheid van je CSS kan verstoren.
