124-125. What Is Bootstrap & Including It

Bootstrap is het populairste CSS-framework ter wereld. Een framework geeft je een verzameling van vooraf gestijlde componenten (knoppen, formulieren, navbars) en een krachtig responsive grid-systeem, zodat je snel mooie, responsive websites kunt bouwen. Het voordeel is snelheid en consistentie. Een potentieel nadeel is dat sites er 'Bootstrap-achtig' uit kunnen zien als je ze niet verder aanpast.

126-128. Bootstrap Components (Buttons, Alerts)

Gebruik de voorgedefinieerde klassen van Bootstrap om snel componenten te stijlen. In plaats van zelf CSS te schrijven voor een knop, voeg je klassen toe zoals btn en btn-primary. Andere veelgebruikte componenten zijn Alerts, Badges, Cards (een flexibele container voor content), en Modals (pop-up vensters).

129-131. The Bootstrap Grid System

Bootstrap heeft een krachtig 12-koloms flexbox grid-systeem. Definieer een rij (.row) en plaats daarin kolommen (.col-*). De som van de kolommen in een rij moet idealiter 12 zijn. Het systeem is mobile-first: de .col-6 klasse geldt voor alle schermgroottes. Responsive klassen zoals .col-md-8 (md voor medium) gelden alleen vanaf dat breekpunt en groter, waardoor je complexe, responsive layouts kunt maken.

132-135. Bootstrap Forms, Navbars & Utilities

Bootstrap maakt ook het stijlen van formulieren en navigatiebalken zeer eenvoudig met specifieke klassen. Een van de krachtigste aspecten zijn de "utility classes". Dit zijn kleine, doelgerichte klassen die één ding doen, zoals het toevoegen van marge (mt-2 voor margin-top), padding (p-3 voor padding), of het centreren van elementen (mx-auto). Deze utilities versnellen het ontwikkelproces enorm, omdat je veel kleine aanpassingen direct in de HTML kunt doen.
